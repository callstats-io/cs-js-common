# cs-js-common
Javascript code for some common utilities to be shared amongst csio projects.

## Usage

```javascript

  var cs_utils = require('cs-js-common');

  cs_utils.isString(anObj);

  cs_utils.stringify(anObj);
```

## Tests

```shell
  npm test
```
